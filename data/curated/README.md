# Curated Catalogue

The MVP target file is:

```text
data/curated/abo_mvp_catalogue.csv
```

Required columns:

- `item_id`
- `normalized_category`
- `title`
- `width_cm`
- `depth_cm`
- `height_cm`
- `material`
- `color`
- `style_text`
- `price_inr`
- `price_note`
- `image_id`
- `image_path`
- `image_available`
- `source_dataset`

Prices must be labelled as curated indicative demo prices because ABO does not
provide product prices.
