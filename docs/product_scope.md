# FormaOS Product Scope

## MVP Goal

FormaOS helps a user turn a room brief into a buildable interior-design plan.
The MVP proves that the output can be grounded in real catalogue items with
prices, dimensions, fit checks, budget checks, and optional Vastu guidance.

## Ships In The MVP

- Multi-page frontend workspace.
- Chat-style brief capture.
- Room setup controls for room type, dimensions, units, budget, style notes,
  and optional Vastu preference.
- ABO-grounded item retrieval from a curated furniture catalogue.
- Fit, budget, sourceability, and optional Vastu checks.
- Shopping list with item names, dimensions, prices, fit notes, and total cost.
- Export/share brief page.
- Evaluation metrics for fit rate, budget accuracy, sourceability, and optional
  Vastu compliance.

## Deferred

- Full 3D room reconstruction.
- CAD-quality floor plans.
- Automatic measurement from photos.
- Human designer marketplace.
- Image inpainting or product-conditioned image editing.
- Production authentication.
- Payments and checkout.
- Complete professional Vastu rule system.

## Demo Must Prove

Given a living-room brief, FormaOS returns a plan where each selected item is a
real catalogue item, has dimensions and an indicative INR price, fits the stated
room constraints, and keeps the plan within the user's budget.

## Non-Claims

FormaOS is not a CAD tool, architectural authority, professional interior
designer, or authoritative Vastu consultant. Vastu is opt-in traditional
guidance surfaced transparently as rules and notes.
